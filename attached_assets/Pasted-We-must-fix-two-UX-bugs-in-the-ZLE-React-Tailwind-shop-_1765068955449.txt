PART 1 — Global ScrollToTop on route change
────────────────────────────────────────

1. Create a helper component:

   client/src/components/layout/ScrollToTop.tsx

   Code (TS or JS depending on project):

   import { useEffect } from "react";
   import { useLocation } from "react-router-dom";

   const ScrollToTop = () => {
     const { pathname } = useLocation();
     useEffect(() => {
       window.scrollTo({ top: 0, left: 0, behavior: "auto" });
     }, [pathname]);
     return null;
   };

   export default ScrollToTop;

2. Inject <ScrollToTop /> INSIDE the router but OUTSIDE page components.

   Example pattern inside App.tsx or Router file:

   <BrowserRouter>
     <ScrollToTop />
     <MainLayout />
   </BrowserRouter>

3. Verify:

   - All other routes behave the same (start at top).
